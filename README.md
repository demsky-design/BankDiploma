Банковская система:
при запуске программы вас встретит 3 пункта - регистрация, вход, выход.
при нажатии на регистрацию вы сможете зарегистрировать себе аккаунт.
при входе войти в него.
выход закрывает программу.
после входа в аккаунт вас встретит 8 пунктов - баланс, снять, пополнить, история, перевод
сменить пинкод, сменить имя, выход из аккаунта.
баланс показывает ваш баланс.
снять - снимает ваши деньги, нельзя снять больше вашего баланса
пополнить - пополняет деньги на ваш счет меньше 0 нельзя
история - показывает историю всего, снятия, пополнения и переводов.
перевод - перевод другому человеку по логину, самому себе нельзя перевести.
сменить пинкод - меняет пинкод.
сменить имя - меняет имя.
выход из аккаунта - перекидывает на меню выше этой
